**Shipped a new, mechanically-distinct game — Skyline** (`games/skyline/`): a
stack-and-slice tower builder. A slab slides above the tower; **drop** it (one control)
and the intersection with the slab below is kept while the overhang is sliced away, so
width is monotonically non-increasing unless you land a **flush** drop (within
`PERFECT_EPS`), which preserves the full width and pays a bonus. Miss entirely → topple.
Speed scales with height. Distinct from all five existing verbs (steer / time-a-catch /
thrust / flip-match / aim-and-bounce) — this is precision-align/stack. Design highlight:
**no timer-driven death** — `tick()` only slides the live slab and can never end the run;
death lives solely in `drop()` on an empty overlap (both pinned by tests). Full quality
bar — pure core (JSDoc'd, seeded RNG for spawn position/direction) + 18 tests (reset,
spawn width invariant, speed cap, edge bounce, perfect/imperfect drops, death boundary,
determinism, milestone range-crossing, tick-only/off-play inertness) + shell (eased
camera, sliced-shard + flush-flash eye-candy, boot-failure fallback, guarded loop) +
