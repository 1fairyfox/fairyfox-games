# 2026-07-13 — GROW: Loft onto varied structure ("the air")

_Newest on top._

## GROW run — Loft gets weather (v0.22.1)

**Picked Loft** as the least-covered game (only Loft and Poise were still off the varied-structure
pattern; Loft was also the flattest run in the collection).

**The problem, stated honestly.** Every other game on the pattern got there by varying its *spawn*
generator. Loft can't: its orbs are **permanent** — nothing spawns except the top-up every 8 points,
and that **caps at six orbs**. So the run had exactly one axis of growth, it ran out early, and after
that every run was the same six orbs in the same dead-still room. Varying "the spawn" would have been
a no-op the player could never feel.

**The call: the varied unit is the AIR, not the orbs.** The thing that can differ run to run in Loft
is the *weather the orbs fall through*. A run is now a seeded sequence of named **currents**, each a
queue of `{ticks, grav, drift}` beats:

| Current | Stage | Notable | What it does |
|---|---|---|---|
| Still | 0 | – | dead calm — the on-ramp and the breather |
| Drift | 0 | – | a slow breeze: tap where the orb is *going* |
| Thermal | 1 | ✔ | ~0.8× gravity — orbs hang; **the greed window** |
| Gust | 1 | ✔ | a hard, short sideways shove |
| Downdraft | 2 | ✔ | ~1.25× gravity — every timed catch is late |
| The Vortex | 3 | ✔ | heavy air **and** a whipping push (crescendo) |

`FORMATIONS` / `pickFormation` / `loadFormation` / `nextAir` copied in shape from Polarity into
Loft's own core (no shared runtime — the game stays liftable). `minStage` gates each, so climbing
the stages opens the pool; a test pins the calm share falling >75% → <40% from Solo to Zero-G.

**The plateau fix (the real growth).** Gravity now rides a smooth asymptote on the score
(`gravScale`, ×1 → ×1.30, never arriving), so a full sky is no longer the ceiling. Guardrail: a
current is only a *multiplier on that earned ramp* — band-clamped and hard-capped (`GRAV_HARD_MAX`),
so no weather can spike past the difficulty the score earned. A test asserts even a rogue
out-of-band current can't break the cap.

**Legibility.** Notable currents flash a quiet `#formCue`; calm air passes silently. And the shell
now draws a field of faint **dust motes** carried by exactly the live current — so a Thermal
(motes hanging) or a Gust (motes streaming sideways) is *visible before it's named*. View-only,
reduced-motion aware (fewer motes).

**Tests:** Loft 31 → 43 (+12): pool well-formed, beats inside the legal bands, `pickFormation`
stage-eligible + deterministic, progression opens the pool, frame-one still-air guard, the beat
queue never empties over a 4,000-tick run, distinct seeds → distinct weather (same seed identical),
`gravScale` never plateaus/never passes its ceiling, the hard cap holds, and Thermal/Downdraft/Gust
are each measurably felt. Collection **493/493** green. The pre-existing self-play winnability test
still passes unchanged under the new physics — a useful sign the tuning didn't get mean.
